Desafio 04 - Calculadora

Algoritmos computacionais

Inicio 

1° Passo = Pedir para o usuário Digitar o primeiro número;
2° Passo = Leia o número digitado n1;
3° Passo = Pedir para o usuário digite o segundo número;
4° Passo = Leia o número digitado n2;
5° Passo = Efetua a soma de n1 + n2;
6° Passo = Exibir o resultado;

Fim do algoritmo
